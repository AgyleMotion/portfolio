Thermally Induced Helical Ribbon Formation

Performed coupled thermo-mechanical Abaqus/Explicit simulations to model thermally induced roll-up and helical coiling of a flat multilayer ribbon stack driven by differential thermal expansion between layers.

Abaqus/Explicit Thermo-Mechanical Coupling Shell Elements Multilayer Structures Helical Actuation

  • Thermally induced roll-up caused by mismatched thermal expansion coefficients
  • Formation of 3D helical geometries from initially flat ribbon stacks
  • Bilinear shell elements used to capture large deformation and coiling behavior
  • Applications in microrobotics and Smart Actuators

Fluid-Solid Interaction Simulation

Simulation of highly elastic human red blood cells (RBCs) in torodial geometry.

  • Coupled Fluid-Solid Interaction.
  • No-slip condition is enforced at the geometry wall using the ghost-node IBM.
  • Finite element is used to compute the elastic tension 𝜏₁, 𝜏₂ in each element of RBC elastic membrane.
  • Each cell is discretized using Delaunay triangles Ne = 5120 elements.
  • RBC is modelled with constitutive hyperelastic model.
  • RBCs are injected in torodial geometry.
  • Simulations is performed on Amarel super computer.
